Fritz
-----
Fritz2 feature tests.

Build
-----
1. gradle clean build

Test
----
1. gradle clean build test

Run JS
------
1. gradle jsRun

Run JVM
-------
1. gradle jvmRun

Curl
----
1. curl -v http://localhost:7979/now

Logs
----
1. build/kotlin.log
2. build/kotlin.test.log
